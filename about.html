Tentang Saya

Saya merupakan mahasiswa Program Studi Sistem Informasi yang memiliki ketertarikan pada pengembangan aplikasi berbasis web dan mobile. Saya terbiasa membangun aplikasi yang bersih, fungsional, dan berorientasi pada pengalaman pengguna.

Saya memiliki pengalaman dalam penggunaan Flutter, Laravel, Firebase, serta pengembangan REST API. Selain itu, saya juga melakukan penelitian analisis sentimen menggunakan Natural Language Processing (NLP) untuk menganalisis engagement media sosial.

Saya senang mempelajari teknologi baru dan terus mengembangkan kemampuan teknis serta pemecahan masalah melalui proyek akademik maupun pengembangan mandiri.
