Tentang Saya

Saya merupakan mahasiswa Program Studi Sistem Informasi yang memiliki ketertarikan pada pengembangan aplikasi berbasis web dan mobile. Saya terbiasa mengerjakan proyek akademik dan mandiri untuk melatih kemampuan teknis serta pemahaman alur pengembangan aplikasi.

Saya memiliki pengalaman dasar dalam pengembangan frontend menggunakan HTML, CSS, dan JavaScript, serta pengembangan mobile menggunakan Flutter dan Dart. Untuk sisi backend, saya terbiasa menggunakan Laravel, MySQL, serta REST API dasar dalam konteks pembelajaran.

Selain itu, saya juga memiliki pengalaman akademik dalam pengolahan data teks dan analisis sentimen menggunakan teknik dasar Natural Language Processing (NLP). Saya terbuka untuk belajar, menerima arahan, dan mengembangkan kemampuan selama program magang atau posisi entry level.
